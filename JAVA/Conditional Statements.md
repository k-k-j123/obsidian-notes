
Java Conditional statements are same as C or C++

### If else:
```
if (condition){
	//code
}
else{
	//code
}
```

### If else if else:
```
if (condition){
	//code
}
else if (condition){ 
	//code
}
else{
	//code
}
```

### ternary operators:
```
variable = condition? statement 1: statement 2
						true case   false case
```

### Switch Statement:
```
swtich(variable){
case 1:
	//code;
case 2:
	//code;
default:
	//code
}
```
